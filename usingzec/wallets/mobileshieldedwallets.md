# Mobile Shielded Wallets

Shielded wallets are wallets that use shielded Zcash addresses (z-addrs). These wallets provide the highest level of privacy. The majority of these wallet are mobile phone apps.

---

Mobile Shielded Wallets

- Nighthawk

- Unstoppable

- Edge

- Ywallet

- Zecwallet Lite
